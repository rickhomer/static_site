HOMER Pro – Differences from HOMER 2

HOMER Energy is very proud to have finally released HOMER Pro. The marketplace pressure to get this new architecture, improved look and feel, modularization, and greatly improved performance was great.

The current version, 3.0.X, however, is missing certain features and components found in HOMER 2. With some very minor exceptions, these features will be available in HOMER Pro 3.1.0, due for release on January 15, 2015. 


HOMER 2 features missing in HOMER Pro:

Advanced Grid Options Full grid capability of HOMER 2 plus new grid features including scheduling and rolling blackouts will be available in 3.1 as an add-on module.
Hydrogen Full hydrogen capability of HOMER 2 will be available in 3.1 as an add-on module.
Linked Sensitivities Link cost sensitivities will be available in 3.1.
Battery Designer Batteries can only be defined manually in the library in HOMER Pro 3.0, i.e. for the kinetic model, the user must specify rate constant, capacity ratio, maximum capacity. HOMER 3.1 will have full battery design capability, as in HOMER 2.
Wind Turbine Designer The HOMER Pro 3.0 capability is very similar to HOMER 2, but it is missing certain features that made things easier in HOMER 2. These features will be in HOMER Pro 3.1.
Copy component from model to library When opening a file, HOMER 2 prompts the user to import new components from the model into the library. HOMER Pro will have a “copy to library” button available in all component menus.
Vanadium redox batteries Full vanadium redox capability of HOMER 2 will be available in 3.1.
Results recycling This is primarily a performance feature – HOMER 2 will only recalculate results for new sensitivity and optimization values, making it easy to gradually change your search space as you learn more. HOMER Pro is considerably faster than HOMER 2. This feature will be implemented in HOMER Pro 3.1, resulting in even greater performance for modelers who like to use this iterative approach.

Because of these missing features, we are (1) offering HOMER Pro as a free upgrade to HOMER 2 users, until January, 15, 2015, and (2) we are selling HOMER Pro at a special launch price through January 15, 2015.This way you can choose which product works best for you, at no additional cost. To get your free matching HOMER Pro license, first download HOMER Pro and install a trial license, which gives you full access to all features. When you return to the website before January 15, 2015, and select to purchase HOMER Pro, you will be offered a free license that will match your HOMER 2 license (it will expire at the same time).

You can run HOMER 2 and HOMER Pro on the same machine at the same time without conflicts. We encourage you to take a look at HOMER Pro, but please review the differences between the products so that you understand the impacts, if any, on your work.
Once HOMER Pro 3.1.0 is released, you can import your HOMER 2 files, if you have decided to continue using HOMER 2.
