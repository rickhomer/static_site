    • Installation and System Requirements

    All current versions of HOMER will install and run on any Windows Operating System. There are no minimum requirements for memory or processors, but these may impact performance and your ability to model complex problems in some cases.

    HOMER will run on a Macintosh equipped with a virtual Windows environment, but we do not provide support for that environment.

    Quantity Discounts

    HOMER 2 Licensing

    Overview

    HOMER 2 uses a serial number/renewal code combination to license a single copy of the software on a single Windows operating system. Purchasing a "license" of HOMER 2 through this website allows you to generate a renewal code that is valid for 180 days after it has been generated. Purchased licenses that have not yet been "used" show up in your history as "unassigned".

    License Term

    Each HOMER 2 renewal code is valid for 180 days after you generate it through the website. You do not have to reinstall HOMER 2 each time you renew. Once your term has expired, find your serial number (Help->About HOMER), generate a renewal code for that serial number through this website, and enter the renewal code into HOMER 2 via the License Management menu item.

    Activating and Assigning HOMER 2 Licenses

    The History->Manage Licenses link lets you generate renewal codes for specific serial numbers. You may also assign a license to another person, if you are a license manager for a group or organization. Assigning a license makes it easier when the assignee requests support. Please be sure to check the serial number of the computer you are licensing before you generate a renewal code. Serial numbers change after a renewal code has been entered, so you need to check this each time you renew. Also, common errors include mixing the number zero (0) and the letter O. Should you accidentally generate a renewal code for an incorrect serial number, you will need to contact customer support to resolve the issue and get a working renewal code.

    Are permanent or enterprise licenses available?

    We have plans to make permanent licenses and enterprise licensing available in new products planned for release in 2014, but HOMER 2 is available strictly on a 6-month basis for a single computer. The manage licenses functionality combined with the bulk discount are meant to provide you with some of the benefits of enterprise or long-term licensing.

    How many people can share a license?

    Licenses are for one computer. HOMER 2 will not work in a server environment. The complete End User License Agreement is here. The quantity discounts that we now offer make it possible for organizations with multiple users to benefit from purchasing licenses in bulk.

    Will my license automatically renew after the 6-month period?

    The license does not automatically renew, and your credit card will not be charged until you renew come to the website and renew. You can begin the renewal process from within the software, and you will be automatically taken to our website where you can purchase a renewal code good for another 6-month period or generate a renewal code from a previously purchased, unassigned license.
